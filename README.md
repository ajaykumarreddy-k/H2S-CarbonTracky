Carbon Tracky allows users to log their transport, dietary, and energy habits, instantly calculate their exact CO₂e footprint, and receive highly personalized, actionable reduction insights generated by **Google's Gemini AI**. 

Eschewing convoluted multi-step forms, Carbon Tracky delivers a true **Unified Single-Page Experience**—an immediate, interactive dashboard that calculates, persists, and intelligently analyzes your carbon data in real-time.

## ✨ Key Features
- **Instant Footprint Calculation**: Input transport, home energy, and dietary metrics to see your footprint in seconds.
- **AI-Powered Personalization**: Stop reading generic advice. Gemini analyzes your specific footprint breakdown and generates targeted, quantified strategies (e.g., *"Switch your 15km petrol commute to transit to save 1,200 kg CO₂e"*).
- **Unified Dashboard**: Live charts, emission histories, global electrical grid intensity, and insights all accessible from one beautifully designed hub.
- **Micro-Animations**: Smooth, professional UI transitions providing a premium, modern feel.

## 🏗️ Architecture & Tech Stack

Carbon Tracky is built using state-of-the-art tools optimized for speed, reliability, and developer experience.

### 🎨 Frontend
- **React + TypeScript**: Fully strongly-typed frontend for robust, error-free UI development.
- **Vite & Bun**: Ultra-fast hot-module replacement and instant dependency installation.
- **Tailwind CSS**: Utility-first styling for a beautiful, responsive, and cohesive design system.
- **Zustand**: Lightweight, frictionless global state management without the boilerplate.

### ⚙️ Backend
- **Python FastAPI**: A blisteringly fast, asynchronous REST API framework.
- **`uv`**: The extremely fast Python package and project manager written in Rust.
- **Google AI Studio (`google-genai`)**: Next-generation LLM integration for generating insights.
- **Pydantic V2**: Lightning-fast data validation and schema generation.

## ⚡ Why It Stands Out

This architecture was explicitly designed to be **production-grade** and highly scalable:
1. **Asynchronous by Default**: FastAPI ensures that AI inference calls, database writes, and external API requests (like fetching live grid intensity) never block the main thread.
2. **Actionable AI**: The backend uses engineered prompts combined with the user's specific mathematical footprint to force Gemini into returning strict, validated JSON arrays of priority-ranked actions.
3. **Type Safety End-to-End**: From Pydantic schemas in the backend to Zod and TypeScript interfaces in the frontend, data flows safely and predictably across the entire stack.
4. **Resilient Dependency Management**: Utilizing `pyproject.toml` and `uv` ensures that the Python environment builds perfectly and consistently in seconds.

## 🚀 Getting Started

To run Carbon Tracky locally, you need both the frontend and backend servers running simultaneously.

### Prerequisites
- **[Bun](https://bun.sh/)** installed for frontend dependencies.
- **Python 3.9+** and **[uv](https://docs.astral.sh/uv/)** installed for backend management.
- A **Google AI Studio API Key**.

### 1. Start the Backend
Navigate to the `Backend` directory, let `uv` sync the dependencies, and start Uvicorn:
```bash
cd Backend
cp .env.example .env
# Open .env and add your GEMINI_API_KEY
uv sync
uv run uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Start the Frontend
In a new terminal window at the root of the project, run the Vite development server via Bun:
```bash
bun install
bun run dev
```

Open `http://localhost:3000` (or the port provided by Vite) in your browser and start tracking!

## 🛣️ Backend API Reference

The FastAPI backend exposes the following RESTful endpoints:

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/calculate` | Computes the exact annual carbon footprint from raw lifestyle data. |
| `POST` | `/api/insights` | Generates personalized Gemini AI reduction strategies based on the calculation result. |
| `POST` | `/api/entries` | Persists a newly calculated footprint and its associated AI insights. |
| `GET` | `/api/entries/{id}` | Retrieves historical footprint entries and insights for dashboard charts. |
| `GET` | `/api/carbon/global` | Fetches real-time global carbon grid intensity (gCO₂/kWh). |
| `GET` | `/api/health` | Returns the operational status of the API and Gemini AI availability. |

*(All routes are automatically proxied via Vite from the frontend).*
